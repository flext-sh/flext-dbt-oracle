# FLEXT dbt Oracle

Projeto dbt para transformar dados Oracle em estruturas analiticas reutilizaveis.

Descricao oficial atual: "FLEXT dbt Oracle - dbt Models for Oracle Database".

## O que este projeto entrega

- Organiza regras de negocio Oracle em modelos dbt.
- Entrega camada analitica para relatorios e indicadores.
- Padroniza transformacao para uso recorrente no portfolio.

## Contexto operacional

- Entrada: datasets Oracle em staging.
- Saida: marts analiticos prontos para consumo.
- Dependencias: flext-db-oracle e ambiente dbt settingsurado.

## Estado atual e risco de adocao

- Qualidade: **Alpha**
- Uso recomendado: **Nao produtivo**
- Nivel de estabilidade: em maturacao funcional e tecnica, sujeito a mudancas de contrato sem garantia de retrocompatibilidade.

## Diretriz para uso nesta fase

Aplicar este projeto somente em desenvolvimento, prova de conceito e homologacao controlada, com expectativa de ajustes frequentes ate maturidade de release.
